# Heart Attack Risk Prediction Analysis

A data mining project focused on identifying and visualizing risk factors associated with heart attack occurrence using the Heart Attack Prediction Dataset.

## Overview

This project provides an automated pipeline for:

- **Data Acquisition:** Dynamic fetching of dataset from GitHub via Raw URL.
- **Exploratory Data Analysis (EDA):** Statistical correlation of lifestyle factors (Cholesterol, Physical Activity, BMI).
- **Preprocessing:** Feature engineering and data cleaning for model readiness.

## Technical Stack

- **Language:** Python 3.12
- **Environment:** Google Colab / Jupyter
- **Primary Libraries:** `pandas`, `numpy`, `matplotlib`, `seaborn`, `scikit-learn`

## Dataset Information

The analysis utilizes the [Heart Attack Prediction Dataset](https://www.kaggle.com/datasets/iamsouravbanerjee/heart-attack-prediction-dataset) from Kaggle. Key features include:

- **Clinical Metrics:** Blood Pressure, Cholesterol, Heart Rate.
- **Lifestyle Factors:** Exercise hours, Smoking status, Alcohol consumption.
- **Demographics:** Age, Sex, Income.

## Execution Instructions

1. Open the [Google Colab Notebook](https://colab.research.google.com/drive/1bwbcnVHnNtBKW5bC5pjqmAryg8_EGSY0).
2. Select **Runtime** > **Run all**.
3. No manual file uploads are required as the data is sourced remotely.
